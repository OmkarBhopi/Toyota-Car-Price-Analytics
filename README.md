# 🚗 Toyota Used Car Price Analytics | Python • SQL • Power BI

## 📌 Project Overview

This project presents an end-to-end data analytics solution for analyzing the Toyota Used Cars dataset. It demonstrates the complete analytics workflow starting from raw data cleaning to business intelligence dashboard development.

The objective is to uncover key business insights affecting used car prices by leveraging Python for data preprocessing and exploratory data analysis (EDA), PostgreSQL for business-oriented SQL analysis, and Power BI for interactive dashboard creation.

This project showcases industry-standard data analytics practices and highlights skills required for Data Analyst and Business Intelligence roles.

---

# 🎯 Objectives

- Perform comprehensive data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Analyze factors affecting used car prices.
- Write business-oriented SQL queries for decision-making.
- Build an interactive Power BI dashboard.
- Generate actionable business insights.

# 💡 Key Business Insights

- Vehicle age has a strong negative relationship with resale price.
- Cars with lower mileage generally command higher prices.
- Horsepower positively influences vehicle pricing.
- Fuel type significantly impacts resale value.
- Premium vehicles retain higher resale prices.
- Automatic transmission vehicles generally have higher average prices.
